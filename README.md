# FitSphere 健身房管理系统

## 一、系统概述

FitSphere 是一款基于 Spring Boot + Vue 3 构建的智能健身房管理系统，采用前后端分离架构设计。系统支持三种角色登录使用：**管理员**、**会员**和**教练**，提供完整的健身房日常运营管理功能。

## 三、功能模块

### 3.1 角色权限体系

| 角色 | 权限范围 | 核心功能 |
|------|----------|----------|
| **管理员** | 系统全部权限 | 人员管理、课程管理、设备管理、数据统计、系统配置 |
| **会员** | 个人权限 | 签到、预约课程、私教预约、健身计划、个人中心 |
| **教练** | 教学权限 | 课堂签到、课程管理、会员管理、私教预约、个人中心 |

### 3.2 核心功能模块

#### 3.2.1 用户认证模块 (`AuthController`)

- 用户注册（支持邮箱验证码验证）
- 用户登录（账号锁定机制，密码错误5次锁定5分钟）
- 密码重置（邮箱验证码验证）
- JWT Token 认证

#### 3.2.2 用户管理模块 (`UserController`)

- 用户信息查询
- 用户信息修改（支持头像上传）
- 密码修改
- 账号解锁

#### 3.2.3 会员管理模块 (`MemberController`)

- 会员信息维护
- 会员卡管理（`FitnessCardController`）
- 会员签到记录查询
- 会员等级管理

#### 3.2.4 教练管理模块 (`CoachController`)

- 教练信息维护
- 教练课程管理
- 教练排班管理（`CoachScheduleController`）
- 教练评价管理

#### 3.2.5 课程管理模块 (`CourseController`)

- 课程信息维护
- 课程预约管理（`ReservationController`）
- 课堂签到管理

#### 3.2.6 签到管理模块 (`CheckinController`)

- 会员自助签到/签退
- 课程签到
- 签到排行榜（累计签到次数、连续签到天数、本月/本周签到次数）
- 签到统计数据
- Excel 导出功能

#### 3.2.7 私教预约模块 (`PrivateCoachingController`)

- 私教预约提交
- 预约审核流程（管理员审核）
- 兑换券支付（`PrivateCoachingCouponController`）
- 教练确认与完成
- 课程评价

#### 3.2.8 设备管理模块 (`EquipmentController`)

- 设备信息维护
- 设备状态管理
- 设备维护记录

#### 3.2.9 健身计划模块 (`FitnessPlanController`)

- 健身计划创建与管理
- 计划分配

#### 3.2.10 数据看板模块 (`DashboardController`)

- 今日签到人数统计
- 当前在馆人数统计
- 本周/本月签到统计
- 课程预约统计
- 私教预约统计

#### 3.2.11 AI 助手模块 (`AiChatController`)

- AI 健身咨询
- 智能问答服务（基于 DeepSeek API）

#### 3.2.12 文件管理模块 (`FileController`)

- 头像上传
- 文件资源访问

## 五、关键特性

### 5.1 安全特性

- **JWT 认证**：无状态认证，Token 有效期 24 小时
- **角色权限控制**：基于 Spring Security 的细粒度权限管理
- **密码加密**：BCrypt 加密存储
- **账号锁定**：登录失败 5 次自动锁定 5 分钟
- **CORS 配置**：支持跨域访问
- **头像路径白名单**：静态资源访问跳过 Token 验证

### 5.2 性能特性

- **Redis 缓存**：教练列表、统计数据等使用 Redis 缓存
- **查询优化**：MyBatis Plus 分页查询、联合查询优化
- **异步邮件**：验证码发送采用异步线程

### 5.3 业务特性

- **签到排行榜**：支持累计签到、连续签到、本月/本周签到统计
- **头像同步**：用户修改头像时自动同步到会员/教练表
- **Excel 导出**：支持签到记录导出为 Excel 格式
- **定时任务**：预约超时处理、支付超时处理

## 八、总结

FitSphere 健身房管理系统是一个功能完整、架构清晰的企业级管理系统，具备以下特点：

1. **多角色支持**：管理员、会员、教练三种角色各司其职
2. **前后端分离**：Vue 3 + Spring Boot 架构，便于维护和扩展
3. **安全性高**：JWT 认证、密码加密、权限控制等多重保障
4. **性能优化**：Redis 缓存、查询优化提升系统响应速度
5. **功能完善**：覆盖健身房日常运营的核心业务场景
6. **用户体验好**：现代化 UI 设计，支持动态背景效果
